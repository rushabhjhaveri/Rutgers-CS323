All the code (*.m) files for each algorithm are stored in the code folder. 

Within the testcases folder, there are two subfolders: 
- testcases: This folder contains five testcases for each algorithm written. 
- testcases_results: This folder contains the program outputs for each corresponding testcase in the testcases folder. 

Running the program: 

Please run programs in the following format: 

cubic_splines(filename)

least_squares(filename)

simpson(filename)

Please make sure the files to be read are in the same directory as the code.
